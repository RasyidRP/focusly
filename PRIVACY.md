# Privacy Policy for focusly.
**Last Updated: April 7, 2026**

Rasyid ("the Developer") built the **focusly.** app as a Free application. This service is provided by the Developer at no cost and is intended for use "as is".

This page is used to inform visitors regarding policies with the collection, use, and disclosure of Personal Information for anyone who decides to use this application.

### 1. Data Collection and Storage
**focusly. operates on a strictly offline-first architecture.** The Developer does not collect, transmit, or store any personal data, telemetry, or analytics on external servers. All user data, including task inputs, completion history, and app preferences, is stored locally on the user's device using an encrypted local database. 

If you uninstall the application, your data is permanently deleted from your device.

### 2. Required Device Permissions
To function as a strict productivity enforcer, the application utilizes specific Android system permissions. These are used purely for local device operations:
* **Foreground Services & WakeLocks:** Used to guarantee exact-time alarm firing and task tracking, even when the device enters deep sleep (Doze Mode).
* **System Alert Windows (Draw Over Other Apps):** Used to bypass background execution limits and display the focus-completion UI over other applications.
* **Notifications:** Used to alert the user of active focus sessions and completed tasks.
* **Vibration:** Used for haptic feedback upon task completion.

None of these permissions are used to track user behavior, monitor other apps, or harvest data.

### 3. Third-Party Services
This application does not contain any third-party ad networks, tracking SDKs, or analytics engines. It does contain a standard external web link to the Developer's "Buy Me a Coffee" page, which opens in your device's default web browser.

### 4. Changes to This Privacy Policy
The Developer may update our Privacy Policy from time to time. Thus, you are advised to review this page periodically for any changes. The Developer will notify you of any changes by posting the new Privacy Policy on this page.
